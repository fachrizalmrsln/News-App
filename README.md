# NewsApp
Exploration for simple NewsApp

# Features
1. Retreving data from https://newsapi.org/ and show it in MainActivity.
2. Searching data to https://newsapi.org/ and show it in SearchActivity.
3. Design Pattern Implementation (MVP).
4. Picasso.
5. SwipeToRefresh.
6. Kotlin.
